Для корректной работы программы необходим numpy, чтобы его установить, запустите файл get_numpt.bat

Двойной клик по файлу mapper sin.py - запуск программы
На выходе получится папка с нужными файлами (по умолчанию 1024х1024)

внутри папки to terrain лежит файл который покрасит карту, для этого нужно переименовать картинку в букву английского алфавита, например: a, b, c, d, e, f, g...
после запуск появится файл с той же буквой и припиской color
